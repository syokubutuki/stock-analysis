# 解析信号と瞬時周波数 — 株価の「今この瞬間」を複素数で捉える

## 序 — ひとつの素朴な問いから

株価チャートを眺めていると、ふと気づくことがある。価格は上がったり下がったりしている——つまり「振動」している。しかしその振動は、振り子のように規則正しくはない。ある日は激しく揺れ、別の日は凪いだ水面のように静まり返る。振幅は刻々と変わり、周期もまた一定ではない。

ここで素朴な問いが生まれる。

> **「今この瞬間、株価はどのくらいの速さで、どのくらいの強さで振動しているのか？」**

フーリエ変換はこの問いに答えてくれない。フーリエ変換は時系列全体を眺めて「平均的にどの周波数が支配的か」を教えてくれるが、「今この瞬間」の周波数については沈黙する。ウェーブレット変換はもう少し親切で、時間と周波数の両方についてある程度の情報をくれる。しかしそこには不確定性原理という根本的な制約がある——時間分解能と周波数分解能は同時には高められない。

では、もっと根本的に考え直してみよう。「瞬時的な周波数」という概念そのものを、数学的に厳密に定義できないだろうか。

その答えが**解析信号**（analytic signal）である。

## 第1章 — 実数の裏側にある影

### 1.1 株価は「射影」にすぎない

私たちが日々目にする株価 $p(t)$ は実数値の時系列である。終値は 3,450円であり、3,450 + 200i 円ではない。しかし、ここであえて奇妙な仮説を立ててみよう。

> **仮説**: 株価の動きの「本当の姿」は複素数の世界にある。私たちが見ている実数の株価は、その複素数の実部——つまり影——にすぎない。

これは突飛な話ではない。物理学では日常的に使われる発想である。電磁波の電場 $E(t) = A\cos(\omega t + \phi)$ を解析するとき、物理学者はこれを複素指数関数 $\tilde{E}(t) = Ae^{i(\omega t + \phi)}$ の実部とみなす。なぜなら、複素表現のほうが振幅 $A$ と位相 $\phi$ を自然に分離できるからだ。

株価にも同じことができる。実数の時系列 $x(t)$（ここでは対数リターンを使う）に対して、その「複素数版」を構成する。それが解析信号である。

### 1.2 解析信号の定義

実数値の時系列 $x(t)$ に対して、その**解析信号** $z(t)$ を次のように定義する。

$$z(t) = x(t) + i \cdot \mathcal{H}[x](t)$$

ここで $\mathcal{H}[x](t)$ は $x(t)$ の**ヒルベルト変換**（Hilbert transform）であり、次の積分で定義される。

$$\mathcal{H}[x](t) = \frac{1}{\pi} \, \mathrm{P.V.} \int_{-\infty}^{\infty} \frac{x(\tau)}{t - \tau} \, d\tau$$

$\mathrm{P.V.}$ はコーシーの主値（Cauchy principal value）を意味する。$t = \tau$ で被積分関数が発散するので、その特異点を対称的に除外して積分を定義する。具体的には、

$$\mathrm{P.V.} \int_{-\infty}^{\infty} \frac{x(\tau)}{t - \tau} d\tau = \lim_{\varepsilon \to 0^+} \left[ \int_{-\infty}^{t-\varepsilon} \frac{x(\tau)}{t - \tau} d\tau + \int_{t+\varepsilon}^{\infty} \frac{x(\tau)}{t - \tau} d\tau \right]$$

この式は一見威圧的だが、やっていることは単純である。$x(t)$ の各周波数成分の位相を $90°$ ずらす——それだけだ。

### 1.3 なぜ90°のシフトなのか

ヒルベルト変換の本質を理解するために、周波数領域で考えてみよう。

$x(t)$ のフーリエ変換を $\hat{x}(f)$ とすると、ヒルベルト変換のフーリエ変換は驚くほど単純な形をしている。

$$\widehat{\mathcal{H}[x]}(f) = -i \cdot \mathrm{sgn}(f) \cdot \hat{x}(f)$$

ここで $\mathrm{sgn}(f)$ は符号関数で、$f > 0$ のとき $+1$、$f < 0$ のとき $-1$ を返す。

$-i$ を掛けるとは何か。複素指数関数 $e^{i\theta}$ に $-i$ を掛けると $e^{i(\theta - \pi/2)}$ になる。つまり位相を $\pi/2$（90°）だけ遅らせる。正の周波数成分は $-90°$ シフトし、負の周波数成分は $+90°$ シフトする。

具体例で見てみよう。$x(t) = \cos(\omega t)$ のヒルベルト変換は、

$$\mathcal{H}[\cos(\omega t)] = \sin(\omega t)$$

コサインを90°遅らせるとサインになる——高校数学の事実だ。したがって解析信号は、

$$z(t) = \cos(\omega t) + i \sin(\omega t) = e^{i\omega t}$$

オイラーの公式そのものである。実部と虚部は90°ずれた「直交ペア」を成し、複素平面上では原点のまわりを一定速度で回転する点になる。

### 1.4 解析信号の周波数領域での意味

解析信号 $z(t)$ のフーリエ変換を計算すると、

$$\hat{z}(f) = \hat{x}(f) + i \cdot \widehat{\mathcal{H}[x]}(f) = \hat{x}(f) + i \cdot (-i \cdot \mathrm{sgn}(f) \cdot \hat{x}(f))$$

$f > 0$ のとき $\mathrm{sgn}(f) = 1$ なので、

$$\hat{z}(f) = \hat{x}(f) + \hat{x}(f) = 2\hat{x}(f)$$

$f < 0$ のとき $\mathrm{sgn}(f) = -1$ なので、

$$\hat{z}(f) = \hat{x}(f) - \hat{x}(f) = 0$$

つまり、

$$\hat{z}(f) = \begin{cases} 2\hat{x}(f) & (f > 0) \\ \hat{x}(0) & (f = 0) \\ 0 & (f < 0) \end{cases}$$

**解析信号とは、負の周波数成分を完全に消し去った信号である。**

これが「解析的」（analytic）と呼ばれる理由でもある。複素解析において、上半平面で正則な関数は負の周波数成分を持たない。解析信号はまさにその性質を持つ。

## 第2章 — 瞬時振幅と瞬時位相

### 2.1 極形式への変換

解析信号 $z(t)$ は複素数だから、直交形式（実部と虚部）から極形式（振幅と位相）に変換できる。

$$z(t) = A(t) \cdot e^{i\phi(t)}$$

ここで、

$$A(t) = |z(t)| = \sqrt{x(t)^2 + \mathcal{H}[x](t)^2}$$

$$\phi(t) = \arg z(t) = \arctan\!\left(\frac{\mathcal{H}[x](t)}{x(t)}\right)$$

$A(t)$ を**瞬時振幅**（instantaneous amplitude）、$\phi(t)$ を**瞬時位相**（instantaneous phase）と呼ぶ。

### 2.2 瞬時振幅の意味 — エンベロープ

$A(t)$ は時系列の「包絡線」（envelope）に相当する。元の信号 $x(t)$ がどれほど激しく振動しているかを、振動そのものを除去して滑らかに表現したものだ。

株価の文脈では、$A(t)$ は**瞬時的なボラティリティの代理変数**と解釈できる。EWMAボラティリティが過去のリターンの加重平均であるのに対し、瞬時振幅は「今この瞬間」の振動の大きさを直接与える。過去の情報を指数的に減衰させるといった恣意性がない。

ただし注意が必要だ。瞬時振幅は「直前の半周期」程度の情報を暗黙に使っている。完全に「今この瞬間」だけの情報ということはありえない。ヒルベルト変換自体が非局所的な演算（全時間の積分）だからである。

### 2.3 瞬時位相の意味 — 振動の「今どこにいるか」

$\phi(t)$ は振動のサイクルにおける現在位置を表す。$\phi = 0$ から $2\pi$ まで進むと1サイクルが完了する。

株価で言えば、瞬時位相は「上昇局面の途中にいるのか、天井付近なのか、下降に転じたところなのか、底に近いのか」を連続的な角度として表現する。

しかしここで技術的な問題がある。$\arctan$ は $(-\pi, \pi]$ の範囲しか返さないので、位相が $\pi$ を超えると $-\pi$ にジャンプする（位相の巻きつき）。これを解消するのが**位相アンラッピング**（phase unwrapping）である。

$$\tilde{\phi}(t) = \mathrm{unwrap}[\phi(t)]$$

アンラッピングは、隣接する位相差 $\Delta\phi = \phi(t) - \phi(t-1)$ が $\pm\pi$ を超えたときに $2\pi$ の整数倍を加減して、位相が単調に進行するようにする操作だ。

### 2.4 瞬時周波数 — 位相の時間微分

いよいよ核心に到達する。**瞬時周波数**（instantaneous frequency）を次のように定義する。

$$\omega(t) = \frac{d\tilde{\phi}(t)}{dt}$$

離散時間では、

$$f(t) = \frac{\tilde{\phi}(t+1) - \tilde{\phi}(t)}{2\pi \cdot \Delta t}$$

$f(t)$ の単位は [cycles/day] である。逆数 $T(t) = 1/f(t)$ は瞬時周期 [days/cycle] を与える。

これこそが冒頭の問いへの答えだ。「今この瞬間、株価はどのくらいの速さで振動しているか」——瞬時周波数 $f(t)$ がそれを教えてくれる。

### 2.5 瞬時周波数の物理的制約

瞬時周波数には物理的に意味のある値を取るための条件がある。Bedrosianの定理によれば、解析信号の瞬時周波数が常に非負（$\omega(t) \geq 0$）であるためには、信号が「狭帯域」（narrow-band）でなければならない。

具体的には、$A(t)$ のスペクトルと $\cos\phi(t)$ のスペクトルが重ならないことが必要とされる。株価のリターン系列は一般にこの条件を満たさないため、生のリターンに直接ヒルベルト変換を適用すると、瞬時周波数が負になったり非物理的に大きくなったりすることがある。

これが、後述する経験的モード分解（EMD）と組み合わせる動機になる。EMDで信号を狭帯域な固有モード関数（IMF）に分解してから、各IMFにヒルベルト変換を適用すれば、瞬時周波数は物理的に意味のある値を取る。このEMD + Hilbert変換の組み合わせがHilbert-Huang変換（HHT）であり、本プロジェクトのEMDChart（`emd.ts`）の延長線上にある。

## 第3章 — 離散実装の詳細

### 3.1 FFTによるヒルベルト変換

連続時間のヒルベルト変換の定義（主値積分）を直接数値計算するのは厄介だが、フーリエ変換を経由すれば驚くほど簡単に実装できる。

**アルゴリズム**:

1. $x[n]$（$n = 0, 1, \ldots, N-1$）のFFTを計算する: $\hat{x}[k]$
2. 片側スペクトルの重みベクトル $h[k]$ を構成する:

$$h[k] = \begin{cases} 1 & k = 0 \text{ (DC成分)} \\ 2 & 1 \leq k \leq N/2 - 1 \text{ (正の周波数)} \\ 1 & k = N/2 \text{ (ナイキスト周波数, Nが偶数の場合)} \\ 0 & N/2 + 1 \leq k \leq N-1 \text{ (負の周波数)} \end{cases}$$

3. $\hat{z}[k] = h[k] \cdot \hat{x}[k]$ を計算する
4. 逆FFTで $z[n]$ を得る

これで $z[n]$ が解析信号である。実部は元の $x[n]$、虚部がヒルベルト変換 $\mathcal{H}[x][n]$ になる。

```typescript
function analyticSignal(x: number[]): { re: number[]; im: number[]; amp: number[]; phase: number[] } {
  const N = x.length;
  // 1. FFT
  const X = fft(x); // 複素数配列 [re, im][] を返す

  // 2. 片側スペクトルの重み
  const h = new Array(N).fill(0);
  h[0] = 1;
  for (let k = 1; k < N / 2; k++) h[k] = 2;
  if (N % 2 === 0) h[N / 2] = 1;
  // h[N/2+1 ... N-1] = 0（負の周波数を消去）

  // 3. 重み付け
  const Z = X.map(([re, im], k) => [h[k] * re, h[k] * im]);

  // 4. 逆FFT
  const z = ifft(Z);

  const re = z.map(([r, _]) => r);
  const im = z.map(([_, i]) => i);
  const amp = z.map(([r, i]) => Math.sqrt(r * r + i * i));
  const phase = z.map(([r, i]) => Math.atan2(i, r));

  return { re, im, amp, phase };
}
```

**計算量**: FFTと逆FFTがそれぞれ $O(N \log N)$ なので、全体で $O(N \log N)$。3年分の日足データ（$N \approx 750$）なら瞬時に終わる。

### 3.2 位相アンラッピング

```typescript
function unwrapPhase(phase: number[]): number[] {
  const unwrapped = [phase[0]];
  for (let i = 1; i < phase.length; i++) {
    let diff = phase[i] - phase[i - 1];
    // 差が π を超えたら 2π を引く、-π を下回ったら 2π を足す
    while (diff > Math.PI) diff -= 2 * Math.PI;
    while (diff < -Math.PI) diff += 2 * Math.PI;
    unwrapped.push(unwrapped[i - 1] + diff);
  }
  return unwrapped;
}
```

### 3.3 瞬時周波数の計算

```typescript
function instantaneousFrequency(unwrappedPhase: number[]): number[] {
  // 中心差分で数値微分（端点は前方/後方差分）
  const N = unwrappedPhase.length;
  const freq = new Array(N);

  freq[0] = (unwrappedPhase[1] - unwrappedPhase[0]) / (2 * Math.PI);
  for (let i = 1; i < N - 1; i++) {
    freq[i] = (unwrappedPhase[i + 1] - unwrappedPhase[i - 1]) / (2 * 2 * Math.PI);
  }
  freq[N - 1] = (unwrappedPhase[N - 1] - unwrappedPhase[N - 2]) / (2 * Math.PI);

  return freq; // 単位: cycles/day。逆数が瞬時周期(日)
}
```

## 第4章 — 何が見えるのか

### 4.1 EWMAボラティリティとの比較

既存の `volatility.ts` ではEWMA（$\lambda = 0.94$）で条件付きボラティリティを推定している。

$$\sigma^2_{\text{EWMA}}(t) = \lambda \cdot \sigma^2_{\text{EWMA}}(t-1) + (1 - \lambda) \cdot r^2(t)$$

これは過去のリターンの指数加重平均であり、$\lambda = 0.94$ という定数が「記憶の長さ」を決めている。過去約17日分（半減期 $= \ln 2 / \ln(1/0.94) \approx 11.2$ 日）のリターンが支配的に効く。

一方、瞬時振幅 $A(t)$ はヒルベルト変換という演算の性質上、信号の局所的な振動構造に適応的に反応する。急激なレジーム変化が起きたとき、EWMAは指数的に追従するのに対し、$A(t)$ はより素早く反応できる。

ただし両者は代替関係ではなく補完関係にある。EWMAは統計的推定量として分散を見ている。$A(t)$ は信号処理的に振動の包絡線を見ている。同じ「ボラティリティ」を異なる数学的言語で記述しているのだ。

### 4.2 瞬時周波数が捉えるもの

瞬時周波数 $f(t)$ は「今、株価がどのくらいの速さで振動しているか」を教える。

- $f(t)$ が高い（瞬時周期が短い）: 高頻度で上下している → 短期の往復が激しい → デイトレーダー的なノイズが支配的
- $f(t)$ が低い（瞬時周期が長い）: ゆっくりした大きな波 → 機関投資家のトレンドフォローやマクロ要因が支配的
- $f(t)$ の急変: 市場のリズムの変化 → レジーム転換のシグナル

### 4.3 位相の極座標表現

解析信号 $z(t) = A(t)e^{i\phi(t)}$ を複素平面上にプロットすると、時間発展が「螺旋」として可視化される。

- 安定した振動: 原点のまわりを一定の半径で回転
- ボラティリティ上昇: 螺旋の半径が拡大
- 周波数の変化: 回転速度の変化

この表現は、後述する「株価ハミルトニアン」や位相同期（PLV）の基礎になる。複素平面上の回転こそが、量子力学的アプローチの出発点である。

## 第6章 — 数学的補足

### 6.1 ヒルベルト変換の基本性質

以下の性質が成り立つ。いずれも厳密な証明は省略するが、直感的説明を付す。

**性質1: 二重適用で符号反転**

$$\mathcal{H}[\mathcal{H}[x]](t) = -x(t)$$

90°のシフトを2回適用すると180°、すなわち符号反転になる。これは $(-i)(-i) = i^2 = -1$ に対応する。

**性質2: エネルギー保存（パーセバルの等式）**

$$\int_{-\infty}^{\infty} |x(t)|^2 \, dt = \int_{-\infty}^{\infty} |\mathcal{H}[x](t)|^2 \, dt$$

ヒルベルト変換は各周波数成分の振幅を変えず位相だけを回すので、全エネルギーは保存される。

**性質3: 直交性**

$$\int_{-\infty}^{\infty} x(t) \cdot \mathcal{H}[x](t) \, dt = 0$$

元の信号とそのヒルベルト変換は直交する。これが解析信号の実部と虚部が「独立な情報」を持つことの根拠である。

### 6.2 解析信号とハーディ空間

厳密に言えば、解析信号は上半平面のハーディ空間 $H^2(\mathbb{R})$ に属する。$H^2$ は「フーリエ変換が正の周波数にのみ台を持つ $L^2$ 関数」の空間であり、

$$H^2(\mathbb{R}) = \left\{ f \in L^2(\mathbb{R}) \mid \hat{f}(\omega) = 0 \text{ for } \omega < 0 \right\}$$

この空間はヒルベルト空間（完備な内積空間）をなす。内積は、

$$\langle f, g \rangle_{H^2} = \int_{-\infty}^{\infty} f(t) \overline{g(t)} \, dt$$

であり、これはまさにエルミート内積である。ここに量子力学との接点がある。量子力学の状態ベクトル $|\psi\rangle$ がヒルベルト空間に住むように、株価の解析信号もヒルベルト空間に住む。「株価ハミルトニアン」の構想は、この数学的構造を真剣に受け取ったものだ。

### 6.3 Bedrosian の定理（瞬時周波数の正当性条件）

**定理** (Bedrosian, 1963): $x(t) = A(t)\cos\phi(t)$ において、$A(t)$ のフーリエ変換の台が $[-f_a, f_a]$ に含まれ、$\cos\phi(t)$ のフーリエ変換の台が $|f| \geq f_a$ に含まれるならば、

$$\mathcal{H}[A(t)\cos\phi(t)] = A(t)\sin\phi(t)$$

が成り立つ。

この定理が成立するとき、解析信号の瞬時振幅は $A(t)$、瞬時位相は $\phi(t)$ に正確に一致する。条件が満たされないと、ヒルベルト変換が振幅変調と位相変調を混同し、物理的に無意味な瞬時周波数が得られてしまう。

EMDの各IMFは（理想的には）この条件を近似的に満たすため、EMD → Hilbert変換の順序が推奨される。

## 結 — 複素数の世界への入口

解析信号という道具立ては、一見すると「ただ虚部を付け加えただけ」に見えるかもしれない。しかしそれは、実数直線上の1次元的な動きを、複素平面上の2次元的な回転運動として捉え直す——次元を一つ上げる——という根本的な視点の転換をもたらす。

この転換によって、「振幅」と「位相」という二つの独立な量が自然に分離され、「瞬時周波数」という概念が厳密に定義される。そしてこの複素数の世界は、次の章で扱うエルミート内積、位相同期、そして「株価ハミルトニアン」への道を開く。

> 株価チャートの裏側には、目に見えない虚部がある。その影を復元することで、振動の「今この瞬間」が初めて見えてくる。
