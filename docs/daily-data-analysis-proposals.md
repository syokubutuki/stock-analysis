# 日足データで追加・拡張できる分析カタログ

本書は、現在取得している**日足データ（日付・始値O・高値H・安値L・終値C・出来高V、ベンチマークも取得可）**だけで実装できる分析を、新規追加・既存拡張の両面から網羅したものである。各項目に **区分**（【新規】/【拡張: 対象コンポーネント】）・**計算根拠**・**投資活用**・**優先度**(★1〜3) を付す。

> 規約: 新規コンポーネントは必ず `AnalysisGuide` で手法・数式・用語・読み方・活用・注意点を解説すること（`CLAUDE.md`）。

---

## 0. 前提 — 日足で「できること / できないこと」

**できる**
- 1本のローソク足の形状（実体・ヒゲ・レンジ）、窓、終値位置
- 始値/終値からの日中・オーバーナイト分解（時刻は不可、配分は可）
- 出来高・流動性プロキシ（高安スプレッド推定、Amihud 等）
- トレンド/モメンタム/平均回帰/ブレイク/季節性/イベント
- ベンチマーク1本との相対力・ベータ・共和分（2系列）
- 長期（最大10年）での持続性検証

**できない（別データが要る）**
- 日中の到達**時刻**（→ 日中足。`HighLowTimingChart` で対応済み）
- 板・約定・ティック由来の真のマイクロストラクチャ
- ファンダメンタルズ・決算日・ニュース・需給（信用残・空売り）
- 多銘柄横断（クロスセクション）ファクター（ユニバース未取得）

## 推奨実装順（Top 10）

実戦性・新規性・実装コストのバランスで:

1. **9.1 状態→先行きリターン表** — 「今この状態で何が起きやすいか」を即提示。最も実戦的。
2. **2.1 OHLCボラ推定量(Yang-Zhang)** — OHLCをフル活用、既存ボラ系の精度向上に波及。
3. **8.1 オーバーナイト vs 日中エクイティ** — 有名な異常を1チャートで。
4. **3.2 高安スプレッド推定(Corwin-Schultz)** — 日足だけで取引コスト/流動性を近似。
5. **5.1 短期リバーサル・エッジ** — 押し目/戻りの定量化。
6. **1.2 レンジ収縮(NR7)→ブレイク** — ブレイク前夜の検出。
7. **11.1 相対力チャート** — 市場対比の強弱、取得済みベンチで即可能。
8. **12.1 ヒストリカル・アナログ** — 直感的で訴求力が高い。
9. **13.1 保有期間別MFE/MAEで最適TP/SL** — 既存MFEMAEの実戦的拡張。
10. **1.1 ローソクパターンの統計的エッジ** — 既存パターン検出に確率の裏付け。

## 備考

- すべて現行の取得データ（10年・日足OHLCV＋ベンチマーク）で実装可能。外部データ追加は不要。
- 「日中の時刻」が要る分析は `HighLowTimingChart`（日中足）側の領分。日足分析は**長期の持続性**と**断面の質**で棲み分ける。
- 各アノマリー系は必ず **サンプル数(n)・有意性・年次持続性** を併記し、過剰解釈を防ぐこと。
