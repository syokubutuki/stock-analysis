## 2. 遅延座標の根本的な問題

### Takensの埋め込み定理が保証すること

Takensの埋め込み定理(1981)は、滑らかな力学系のアトラクタが、1次元の観測量から遅延座標で位相的に同値に再構成できることを示す。具体的には:

- 元の力学系の状態空間が d 次元アトラクタを持つ場合
- 埋め込み次元 m >= 2d + 1 で generic に埋め込みが成立する
- 再構成されたアトラクタは元のアトラクタと微分同相

### 株価時系列に適用する際の困難

1. **非定常性**: 株価の生成メカニズム自体が時間とともに変化する(構造変化、制度変更、参加者の変化)。Takensの定理は定常的な力学系を前提とする。
2. **高ノイズ**: 観測ノイズだけでなく、力学的ノイズ(確率的撹乱)が大きい。アトラクタ構造がノイズに埋もれる。
3. **高次元の可能性**: 市場を駆動する変数(金利、為替、心理、政策、...）は膨大で、有限次元の決定論的アトラクタが存在する保証がない。
4. **データ長の制限**: 高次元の埋め込みには指数関数的にデータが必要。日足データでは数年分でも統計的に不十分なことが多い。

### それでも遅延座標が有用な場面

- **短期〜中期のレジーム内**では、局所的に低次元の構造が現れることがある
- **構造の有無自体が情報**: アトラクタが見えない(= 球状の雲)ならランダム性が支配的 → 予測困難という判断
- **リカレンス構造**: 完全なアトラクタ再構成ができなくても、位相空間上の再帰性(リカレンス)から有用な指標が得られる

## 3. アプローチ1: 遅延座標を改良して使う

### 3.1 τの自動選択 — 自己相互情報量 (AMI)

**現状**: τはスライダーで1〜20の手動選択。根拠なし。

**改善**: 自己相互情報量(Auto Mutual Information)の最初の極小値を最適τとする。

```
原理:
  AMI(τ) = Σ p(x(t), x(t+τ)) * log[ p(x(t), x(t+τ)) / (p(x(t)) * p(x(t+τ))) ]

  τが小さすぎる → AMIが高い → 座標軸間の冗長性が大きい → 対角線に潰れる
  τが大きすぎる → AMIがゼロに近い → 座標間の関連が失われる → 構造が消える
  最初の極小   → 独立性と関連性のバランスが最適
```

**実装方針**: ヒストグラムベースのMI推定(既存のエントロピー計算関数を活用可能)

**投資判断への接続**: 最適τ自体は直接的なシグナルではないが、後続の全ての解析(RQA、Lyapunov、TDA等)の精度を底上げする基盤。

### 3.2 偽最近傍法 (FNN) による埋め込み次元の決定

**現状**: 2Dか3Dの手動選択。

**改善**: False Nearest Neighbors法で最小十分埋め込み次元を決定。

```
原理:
  埋め込み次元 d で最近傍だった点対が、d+1 次元に上げたとき大きく離れる
  → その点対は「偽の近傍」(低次元への射影による見かけの近さ)

  FNN率:
    FNN(d) = (d次元での近傍のうち、d+1次元で閾値以上離れるものの割合)

  FNN率がゼロに近づく最小の d が適切な埋め込み次元
```

**投資判断への接続**:
- FNN率が低次元(d=3〜5)で急減する → **低次元の決定論的構造が存在** → 短期予測が有効な可能性
- FNN率が高次元でも下がらない → **ランダム性が支配的** → テクニカル分析の信頼性が低い
- **FNN飽和次元そのものが銘柄の「複雑さ」の指標**になる

### 3.3 局所Lyapunov指数の時系列化

**現状**: `estimateLyapunov()`は全期間で1つの値を返す。

**改善**: ローリングウィンドウで局所Lyapunov指数を時系列として追跡する。

```
ローリング局所Lyapunov:
  窓幅 W (例: 60〜120日)
  各時点 t で [t-W, t] の部分系列からLyapunov指数を推定

  λ_local(t) > 0  → カオス的・不安定 → 予測困難、ブレイクアウトの可能性
  λ_local(t) < 0  → 収束的・安定 → 引力圏内、平均回帰的
  λ_local(t) ≈ 0  → 臨界状態 → レジーム転換の前兆
```

**投資判断への接続**:
- λ_localが負から正に転じる → **安定状態から不安定への遷移** = ボラティリティ拡大の前兆
- λ_localが正から負に転じる → **カオスから秩序への遷移** = トレンド確定・ボラ収縮
- λ_localの急変自体がシグナル(変化率の閾値でアラート)

### 3.4 位相空間密度による「現在位置」の文脈化

**現状**: 彗星アニメーションが視覚的にやっているが定量化されていない。

**改善**: 現在の埋め込みベクトル周辺の局所密度を計算し、数値化する。

```
局所密度:
  ρ(t) = (半径 ε 内の過去の埋め込みベクトルの数) / (全ベクトル数)

  ρ(t) が高い → よく訪れる領域にいる → 過去のパターンが参考になる
  ρ(t) が低い → 未知の領域にいる → 過去の経験則が通用しない異常事態
```

**投資判断への接続**:
- 高密度領域 → 統計的に安定した判断が可能 → 通常の戦略を適用
- 低密度領域 → **リスク管理を強化**(ポジション縮小、ストップ幅拡大)
- 密度の急減 → アトラクタからの逸脱 = レジーム転換の可能性

## 4. アプローチ2: 遅延座標を使わずにアトラクタ的構造を検出する

### 4.1 Kramers-Moyalポテンシャル関数

遅延座標を経由せず、**株価がどの価格帯に引き寄せられるか**を直接推定する。

```
確率微分方程式:
  dp = μ(p) dt + σ(p) dW

  μ(p): ドリフト関数 — 各価格帯での期待変化方向
  σ(p): 拡散関数 — 各価格帯でのボラティリティ

ポテンシャル関数:
  V(p) = -∫μ(p) dp

  V(p) の極小 = アトラクタ (安定均衡価格)
  V(p) の極大 = リペラー (不安定均衡 = ブレイクポイント)
```

**推定方法**:
- 価格をビンに分割し、各ビンでの条件付き期待変化(1次Kramers-Moyal係数)を計算
- μ(p) = E[Δp | p] / Δt (非パラメトリック推定)
- σ²(p) = E[(Δp)² | p] / Δt (2次係数)

**投資判断への接続**:
- 現在の価格がポテンシャルの**谷**にいる → 平均回帰的 → 逆張り有効
- 現在の価格がポテンシャルの**山の上**にいる → 不安定 → 方向性ブレイクを待つ
- ポテンシャルの**形状変化**を追跡 → 安定均衡の消滅/出現 = 構造的レジーム変化
- σ(p)の価格依存性 → レバレッジ効果(下落時にvolが上がるか)の直接検出

**利点**: 遅延座標の次元・τの問題を完全に回避。結果が価格スケールで直接解釈できる。

### 4.2 ローリングRQA指標の時系列化

**現状**: リカレンスプロットは全期間の集計統計を表示。

**改善**: RQA指標をローリングウィンドウで時系列として計算し、株価チャートにオーバーレイする。

```
ローリングRQA:
  窓幅 W (例: 100日)
  各時点 t で [t-W, t] の部分系列からRQA指標を計算

  追跡する指標:
    DET(t)  — 決定性: 対角線構造の割合
    LAM(t)  — 層状性: 垂直線構造の割合
    TT(t)   — トラッピング時間: 平均滞留時間
    ENTR(t) — 対角線長エントロピー: 対角線長分布の複雑さ
    Lmax(t) — 最長対角線: 最も長い決定論的構造
```

**投資判断への接続**:
- **DETの急減** → 予測可能な構造の崩壊 → レジーム転換のシグナル → ポジション縮小
- **LAMの急増** → 状態の「固着」 → 強いトレンドの持続(ブレイクアウト後の継続)
- **TTの増加** → 一つの状態に留まる時間が伸びている → モメンタム戦略に有利
- **DETとLAMの乖離** → DET低・LAM高 = 予測不能だが固着している = ボラティリティ急変の前兆
- **ENTRの急増** → 対角線パターンの多様化 → 市場の不確実性増大

### 4.3 TDAベッティ数のローリング監視

**現状**: パーシステントホモロジーは全期間で1回計算。

**改善**: ローリングウィンドウでベッティ数の変化を追跡する。

```
ローリングTDA:
  窓幅 W (例: 120日)
  各時点 t で [t-W, t] の部分系列からTDAを計算

  追跡する指標:
    β₀(t) — 連結成分数 → クラスターの数 → レジーム数
    β₁(t) — ループ数 → 周期的構造の数
    total_persistence(t) — 全パーシステンスの合計 → 構造の頑健さ
```

**投資判断への接続**:
- **β₁の増加** → 周期的パターンの出現 → オシレーター系戦略(RSI、ストキャスティクス等)が有効化
- **β₀の増加** → クラスター分裂 → 市場が複数レジームに分散 → リスク増大
- **total_persistenceの急減** → 位相的構造の消失 → ノイズ支配 → 系統的戦略の信頼性低下

## 5. アプローチ3: 位相空間予測

### Sugihara-May法 (Simplex Projection / S-map)

Takens埋め込み空間上で**直接的な短期予測**を行う。遅延座標の最も実用的な応用。

```
Simplex Projection:
  1. 現在の状態ベクトル v(t) を埋め込み空間で構成
  2. 過去の全状態ベクトルから v(t) のk近傍を探索
  3. その k 個の近傍が次のタイムステップでどこに移動したかを追跡
  4. 距離の逆数で加重平均 → 予測値 v̂(t+1)

S-map (局所線形予測):
  1. 現在の状態の近傍で局所的な線形モデルをフィッティング
  2. 重み = exp(-θ * d_i / d̄) で、θが局所性のパラメータ
  3. θ=0 で全体線形(AR)、θ→∞ で最近傍のみ使用
  4. 最適θの選択 → 非線形性の強さの指標にもなる
```

**投資判断への接続**:
- **予測値そのもの** → 短期的な方向の見通し(ただし精度に限界あるので単独シグナルとしては弱い)
- **予測スキル(相関係数)の時間変化** → 「今の市場は予測しやすいか」のメタ指標
  - スキルが高い期間 → 系統的戦略の信頼性が高い → レバレッジ可能
  - スキルが低い期間 → ランダム性が支配的 → 待機/リスク縮小
- **最適θの時間変化** → 非線形性の強さの変化 → 線形モデル(AR/MA)がいつ通用するかの判断
- **Simplex vs S-mapの比較** → 非線形性の統計的テスト(Sugihara & May, 1990)

## 7. 各手法の投資シグナルまとめ

### レジーム転換シグナル (リスク管理)

| 指標 | シグナル | 解釈 | アクション |
|------|---------|------|-----------|
| DET(t) 急減 | 決定性の崩壊 | 予測可能な構造が壊れた | ポジション縮小 |
| λ_local(t) 正→負 | カオス→秩序 | 新しい安定構造の形成 | トレンドフォロー開始 |
| λ_local(t) 負→正 | 秩序→カオス | 安定構造の崩壊 | リスクオフ |
| Fisher-Rao距離スパイク | 分布の急変 | レジーム転換 | 既存戦略の再評価 |
| 位相空間密度 急減 | 未知領域への突入 | 過去パターンが無効化 | ストップ幅拡大 |

### 方向性シグナル (エントリー)

| 指標 | シグナル | 解釈 | アクション |
|------|---------|------|-----------|
| KMポテンシャル谷の位置 | 安定均衡価格の特定 | 価格が引き寄せられる先 | 谷に向かう方向にポジション |
| KMポテンシャル山越え | 不安定均衡の突破 | 次の安定点までのブレイクアウト | 突破方向に順張り |
| Simplex予測値 | 短期方向予測 | 位相空間上の軌道延長 | 予測方向にポジション(スキル高い時のみ) |
| LAM(t) 高水準持続 | 状態の固着 | 現在のトレンドが継続 | トレンドフォロー維持 |

### メタ指標 (戦略選択)

| 指標 | 値域 | 解釈 | 推奨戦略 |
|------|------|------|---------|
| 予測スキル(Simplex) | 高 | 短期予測が有効 | アクティブトレード |
| 予測スキル(Simplex) | 低 | ランダム性支配的 | パッシブ/待機 |
| FNN飽和次元 | 低(3-5) | 低次元の決定論的構造 | テクニカル分析有効 |
| FNN飽和次元 | 高(>10) | 高次元/ランダム | ファンダメンタルズ重視 |
| β₁(ループ数) | 増加 | 周期的パターン出現 | オシレーター系戦略 |
| S-map最適θ | 大 | 非線形性が強い | 非線形モデルが有効 |
| S-map最適θ | 小 | 線形性が支配的 | ARモデル等で十分 |
